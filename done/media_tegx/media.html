Тег <picture>дает веб-разработчикам больше гибкости в определении ресурсов изображения.
Элемент <picture> содержит два разных тега : один или несколько тегов <source> и один тег <img>.
Элемент <picture> работает аналогично элементам <video> и <audio>. Вы устанавливаете разные источники,
и первым источником, который соответствует предпочтениям, является тот, который используется.

Тег <optgroup> используеться для формирования опций внутри элемента <select>

Тег <video> используется для внедрения на веб-страницу видео контента.
Для указания источника используется атрибут src или же несколько src обьедененных
в элемент <source>. В таком случае браузер сам определит лучший источник.

Атрибуты

  • poster указывает на катртинку которая будет демонстрироваться до воспроизведения видео
  • src указывает на источник видео
  • width указывает ширину поля воспроизведения
  • height указывает высоту поля воспроизведения
  • loop зацикливает демонстрацию видео
  • controls получает в браузере вид плеера

Атрибут controls позволяет получать в браузере вид аудиоплеера.
Атрибут autoplay позволяет включать автоматическое воспроизведение при загрузке страници.
Атрибут loop позволяет делать воспроизведение циклическим.
Атрибут mute позволяет убирать звук.

Какие возможности открывает использование тега <iframe> <frame>
Тег <iframe> создает плавающий фрейм, который находится внутри обычного документа,
он позволяет загружать в область заданных размеров любые другие независимые документы.
Тег <iframe> является контейнером, содержание которого игнорируется браузерами,
не поддерживающими данный тег. Для таких браузеров можно указать альтернативный текст,
который увидят пользователи. Он должен располагаться между элементами <iframe> и <iframe>.
