Приведите пример использования метода splice как замену для push, pop, shift, unshift

solice это метод с широкими возможностями,он может и добавлять и вычитать элемент он имеет три аргумента 1- индекс 2 - с какого элемента удалять 3 - номера элементов в зависимости от того что нужно они изменяються

push этот метод добавляет один или более элементов в конец массива и возвращяет новую длинну массива для добавления одного или более элементов в конец массива методом splice необходимо в качестве аргументов указать первым индексом число больше длиины текущего массива вторым аргументом 0 не удалять никаких элементов третим один или несколько элементов которые мы хотим добавить.

pop этот метод удаляет последний элемнт из массива и возвращяет его значение для удаления последнего элемента из массива метод splice необходимо в качестве аргументов задать следующие параметры, первым значение предпоследнего элемента массива, вторым 1, тоесть удалить один элемент.

shift этот метод удаляет первый элемент из массива и возвращяет его значение для удаления первого элемента массива методлм splice нам необходимо в качестве первого параметра задать 0 а в качестве второго 1 тоесть начиная с первого жлемента массива удалить один элемент

unshift добавляет один или более элементов в начало массива и возвращяет новую длинну массива для реализации данной операции методом splice необходимо в качестве первого аргумента поставить 0 в качестве второго 0 и дальше один или несколько элементов которые мы хотим добавить.
